# Small Business Automation System

## Project Overview
This project demonstrates how Python and SQL can be used to automate client intake tracking and reporting for small businesses and nonprofit organizations.

## Business Problem
Small businesses often manage client information manually using spreadsheets, which can lead to errors, duplicated work, and slow reporting.

## Solution
This automation pipeline:
- Loads client intake data from a CSV file
- Cleans and standardizes the data
- Stores the data in a SQLite database
- Generates a weekly summary report

## Tools Used
- Python
- Pandas
- SQLite
